# Zeepredict

A modern sports prediction platform.

## About
Zeepredict provides accurate sports match predictions, betting tips, statistics, and insights for football and other sports.

## Features (Planned)
- Match predictions & probabilities
- Live scores and stats
- Betting tips
- Historical data analysis
- User dashboard

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Later: Backend (Node.js/Python), APIs (football-data.org, etc.)
